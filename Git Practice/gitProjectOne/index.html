Git Practice

Basic Git Command

  • git status:
    Check the status of your working directory.
  • git add :
    Stage changes for commit.
  • git commit -m "message":
    Save changes with a message.
  • git log:
    View the commit history.
  • git diff:
    See changes in your working directory.
